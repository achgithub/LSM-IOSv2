PREDICTOR

Score-prediction leagues, without the spreadsheet.

Run a prediction league with friends, family or the office — players call the scoreline for each fixture, the app works out the points and keeps the table up to date.

Predictor is a mode inside the Last Stand Manager app, so it shares the same one-off setup, subscription tiers and optional cloud features — including publishing a live standings page your players can check without needing the app themselves.
