KILLER

Earn lives, then fight to be the last one standing.

Run a Killer pool with friends, family or the office — players build up lives in the Build Phase, then in the Kill Phase every prediction also fires a Hit at an opponent of their choice. Get it wrong and you can lose lives from both sides at once. Last player with lives left wins.

Killer is a mode inside the Last Stand Manager app, so it shares the same one-off setup, subscription tiers and optional cloud features — including sending players a personal link to submit their own build/hit choices each round.
