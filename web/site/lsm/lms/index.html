LAST MAN STANDING

Pick a winner each round. Get it wrong, you're out.

Run a last-man-standing pool with friends, family or the office — everyone picks a team each round, pick wrong (or not at all) and you're eliminated. The app tracks picks, results and eliminations for you.

Last Man Standing is a mode inside the Last Stand Manager app, so it shares the same one-off setup, subscription tiers and optional cloud features — including sending players a personal link to submit their own picks each round.
